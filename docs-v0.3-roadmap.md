# SALTA v0.3.0 Roadmap

## alpha.1 — Rooms and credentials
- Room entities and device assignments
- Global Shelly credentials
- Per-device credential inheritance and overrides
- Encrypted secret storage

## alpha.2 — Shelly clients
- Gen1 HTTP client
- Gen2/Gen3/Gen4 RPC client
- Authentication resolver
- Connection tests and diagnostics

## alpha.3 — Discovery
- Manual host onboarding
- mDNS discovery
- Device generation and capability detection

## beta.1 — Control
- Switches, outlets, lights, covers and energy meters
- Unified capability mapping
- Retry, timeout and offline handling

## beta.2 — Live dashboard
- Live state transport
- Purpose-built device controls
- Room overview and adapter diagnostics

## rc.1 — Stabilization
- Migrations, backup/restore verification, tests and documentation
